BakLeezzard
===========

주제 : 지하철어플

아이디어 :

어플 시작 후 첫 화면은 주변역검색결과목록, 즐겨찾기목록, 최근검색목록이 출력되어 선택할 수 있다.
역을 검색하고 싶다면 main버튼을 눌러서 역 검색 화면으로 이동한다.
역 검색 화면에는 위에서부터 역검색창, 노선도가 배치되어 있으며 검색창이나 노선도를 이용하여 역 지정 후 출발역과 도착역으로 설정하여 경로를 볼 수 있으며 해당 역의 정보를 볼 수도 있다.
출발역과 도착역을 설정하여 경로를 검색하여 즐겨찾기로의 등록과 알람설정을 할 수 있다.
역의 졍보에는 열차정보(이전역, 다음역, 해당역의 첫차와 막차시간, 도착예정시간)와 해당 역의 전체 시간표, 역정보(주소, 연락처)가 포함되어 있다.
